# node-user-management
User management system built with Node.js, Express and SQLite.

This project implements a complete CRUD for users with:
- Form-based data entry
- REST API
- Frontend rendered with vanilla JavaScript (Fetch + DOM)
- Server-side validations
- Modular backend architecture

Planned features:
- Authentication & authorization
- Role-based access control (admin/user)
- Route protection
- Input hardening and security best practices
